## Tools Used

### 1. Python
Python is the primary language used in Artificial Intelligence because of:
- simple syntax
- large ML ecosystem
- community support

We install Python and verify installation:

```bash
python --version
```

### 2. VS Code
VS Code acts as our development environment where we:
- write code
- run scripts
- debug errors
- manage extensions

### 3. Important Extensions

- Python (Microsoft)
